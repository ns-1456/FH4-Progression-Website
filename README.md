# The Gentleman's Tour | FH4 Companion

A beautiful, immersive companion website for tracking your progress through Forza Horizon 4's "Gentleman's Tour" campaign.

## Features

- **Era-Based Progression**: Track your journey through 6 distinct eras (1950s â†’ Hypercars)
- **Automatic Progress Saving**: All progress is automatically saved to your browser's local storage
- **Auto-Advance**: When you complete all objectives in an era, the next era automatically unlocks
- **Visual Progress Tracking**: See your completion percentage and current era at a glance
- **Organized Categories**: 
  - Garage (cars to acquire)
  - Milestones (racing levels and achievements)
  - Challenges (special objectives)
  - Scrapbook (photo opportunities)
  - Legends (rare collectible cars)
- **AI Mechanic**: Get tuning advice powered by Gemini AI (requires API key)

## Usage

Simply open `index.html` in your web browser. Your progress is automatically saved locally and will persist between sessions.

### Resetting Progress

Click the "Reset Campaign Data" link at the bottom of the page to start fresh.

## Setup (Optional - AI Features)

To enable the AI mechanic features, you'll need to add a Gemini API key to the `apiKey` variable in the JavaScript section.

## Technologies

- Pure HTML/CSS/JavaScript
- LocalStorage for data persistence
- Font Awesome icons
- Google Fonts (Montserrat, Orbitron)
- Gemini AI API (optional)
